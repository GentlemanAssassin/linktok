# LinkTok

A swipe-based link triage app — save articles, triage them TikTok-style.

## Deploy to GitHub Pages

```bash
# 1. Create repo and push (requires gh CLI)
gh repo create linktok --public --push --source=.

# 2. In GitHub: Settings → Pages → Source → GitHub Actions
# 3. Push any change to trigger deploy, or re-run the workflow manually
```

Live at: `https://YOUR-USERNAME.github.io/linktok/`

## Local preview

Just open `index.html` in a browser.
